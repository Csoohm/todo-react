# 📝 To-Do List App

A simple, clean task management web app built with React and Vite.

**Live Demo:** https://csoohm.github.io/todo-react/

## Features

- ✅ Add, edit, and delete tasks
- ✅ Mark tasks as complete with a timestamp
- ✅ Delete all tasks at once (with confirmation)
- ✅ Persistent storage using `localStorage` — tasks stay saved across page refreshes and browser sessions
- ✅ Responsive UI built with Bootstrap

## Tech Stack

- **Frontend:** React, Vite
- **Styling:** Bootstrap
- **Storage:** Browser LocalStorage
- **Deployment:** GitHub Pages

## Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/Csoohm/todo-react.git
cd todo-react/todo-react
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Deployment

This project is deployed to GitHub Pages using `gh-pages`:

```bash
npm run deploy
```
